# Pixel Fate: Arcane Tarot (像素命运：奥术塔罗)

一个基于 React 和像素艺术风格构建的现代塔罗牌占卜与互动应用。结合了暗黑奥术美学与流畅的 UI 交互，为用户提供沉浸式的塔罗体验。

## 🌟 核心功能

- **沉浸式占卜 (Ritual)**：
  - 经典的塔罗牌抽取流程，包含洗牌动画与仪式感十足的视觉特效。
  - 支持四元素卡槽（火、风、水、土），模拟深度的塔罗布局。
  - 智能 MBTI 倾向分析，根据抽取的卡牌与正逆位状态，推导出用户的性格色彩。
- **全动态卡牌系统**：
  - 包含完整的 78 张 Rider-Waite 塔罗牌（22 张大阿卡纳 + 56 张小阿卡纳）。
  - **动态视觉等级**：卡牌拥有不同的视觉等级（普通、卓越、传奇），随着等级提升，卡面会增加动态金线、辉光与材质细节。
  - **暗金发光质感**：精心设计的卡牌背面，采用暗金配色的复杂纹路，并带有动态呼吸发光效果。
- **互动虚拟形象 (Arcane Avatar)**：
  - 内置像素风格的神秘学引导者。
  - **动态情绪系统**：根据用户的操作（如长时间不操作、抽到高阶卡牌等）触发不同的对白与表情。
  - **实时粒子特效**：虚拟形象周围环绕着与其等级相对应的奥术粒子。
- **多维度系统管理**：
  - **图书馆 (Library)**：查看已收集的卡牌及其详细含义。
  - **图鉴 (Codex)**：查看卡牌的视觉演化等级。
  - **历史记录 (History)**：保存并回溯每一次占卜的结果。
  - **动态指南 (Guide)**：集成式的游戏操作手册。

## 🎨 视觉风格

- **暗黑像素美学**：采用深靛蓝与暗紫色的基调，搭配高对比度的发光元素（如霓虹蓝、暗金色）。
- **毛玻璃质感 (Glassmorphism)**：所有 UI 面板均采用半透明冷色调冷磨砂玻璃效果，增强层次感。
- **奥术粒子特效**：基于 Framer Motion 实现的高性能背景星尘与奥术涟漪效果。

## 🛠️ 技术栈

- **核心框架**：[React 19](https://react.dev/)
- **开发构建**：[Vite](https://vite.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **样式方案**：[Tailwind CSS 4.0](https://tailwindcss.com/) (现代化的实用类样式)
- **动画引擎**：[Framer Motion](https://www.framer.com/motion/) (用于复杂的布局过渡与粒子动画)
- **数据持久化**：LocalStorage (用于保存用户卡牌等级与历史记录)
- **资源代理**：wsrv.nl (用于跨域加载并优化卡牌图片资源，解决 ORB/CORS 问题)

## 🚀 快速启动

### 准备环境
确保已安装 [Node.js](https://nodejs.org/) (建议 v18+)。

### 安装依赖
```bash
npm install
```

### 运行开发服务器
```bash
npm run dev
```

### 构建项目
```bash
npm run build
```
